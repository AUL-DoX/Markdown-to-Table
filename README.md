# Markdown ➡ テーブル変換くん v4.0

Markdown表・CSV・TSV・スプレッドシート貼り付けを、**noteに貼りやすい表画像 / グラフ画像** に変換するシングルHTMLツールです。

`Markdown ➡ テーブル変換くん v4.0` は、ブラウザだけで動きます。サーバー不要で、表とグラフをプレビューしながら、そのまま PNG 保存またはクリップボードコピーできます。

## 主な機能

- Markdown表、CSV、TSV、コピペ表の読み込み
- 表モードとグラフモードの切り替え
- PNG保存
- 画像のクリップボードコピー
- note向けの横長、正方形、縦長サイズ対応
- 棒グラフ、折れ線、エリア、集合棒、積み上げ棒、散布図、円グラフに対応
- 表ヘッダー色、グラフ主色、副色、補助線色、軸色のカスタム変更
- 単一HTMLで動作

## 画面イメージ

- 入力データを左側に貼り付ける
- 出力モードを `表` または `グラフ` に切り替える
- タイトルや色を調整する
- 右側プレビューを確認して `PNGで保存` または `画像をクリップボードへコピー`

## 使い方

### 1. ローカルで使う

`note-chart-maker.html` をブラウザで開くだけで使えます。

### 2. データを貼り付ける

以下のような形式に対応しています。

```md
| 月 | 売上 | 目標 |
|---|---:|---:|
| 1月 | 120 | 100 |
| 2月 | 135 | 110 |
| 3月 | 168 | 130 |
```

CSV でも使えます。

```csv
月,売上,目標
1月,120,100
2月,135,110
3月,168,130
```

### 3. 出力モードを選ぶ

- `表`: Markdown表をそのまま見やすい表画像にする
- `グラフ`: 数値列をもとにグラフ画像にする

### 4. 色やレイアウトを整える

- 表ヘッダー背景色
- 表ヘッダー文字色
- グラフ主色
- グラフ副色
- 補助線色
- 軸色
- タイトル、補足、画像サイズ

### 5. 画像を書き出す

- `PNGで保存`
- `画像をクリップボードへコピー`

## GitHub Pages で公開する方法

GitHub Pages でトップページとして開きたい場合は、`note-chart-maker.html` を `index.html` にリネームして公開するのがいちばん簡単です。

### 最小構成

```text
repo-root/
├─ index.html
└─ README.md
```

### 公開手順

1. GitHub にリポジトリを作成する
2. `note-chart-maker.html` を `index.html` に変更して配置する
3. `README.md` を同じリポジトリに置く
4. GitHub の `Settings` → `Pages` を開く
5. `Deploy from a branch` を選ぶ
6. 対象ブランチと `/ (root)` を選ぶ
7. 保存後、数分待つ

公開後は以下のようなURLでアクセスできます。

```text
https://<your-account>.github.io/<your-repository>/
```

## 注意点

- PNG出力は `html2canvas` を使っています
- クリップボードへの画像コピーは、ブラウザによって挙動差が出る場合があります
- GitHub Pages 上でも基本的に動きますが、ブラウザの権限制約でコピー機能の挙動が変わることがあります
